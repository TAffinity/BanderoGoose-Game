# 🐦 BanderoGoose (Python Game)

BanderoGoose is a simple 2D arcade game built with Python and Pygame.  
The player controls a character, avoids enemies, collects bonuses, and tries to get the highest score.

---

## 🎮 Gameplay
- Move the player using arrow keys
- Avoid enemies coming from the right side
- Collect bonuses to increase your score
- The game ends when you collide with an enemy

---

## 🛠️ Technologies
- Python 3.11+
- Pygame
- Random module (for enemy/bonus generation)
- OOP-style game structure (basic)
